# java-GUI-ToDo-App

## To-Do List App (Java Swing)

A simple and clean Java GUI To-Do List application built using Swing.

Users can easily add and delete tasks from a list using buttons.

##Objective

Build a To-Do List using Java Swing with:

JFrame

JButton

JTextField

Add / Delete tasks functionality

## Features

✔ Add tasks

✔ Delete selected tasks

✔ User-friendly GUI

✔ Scrollable task list
